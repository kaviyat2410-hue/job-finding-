# JobConnect-Job portal website
        A responsive job portal website using HTML and css. This project provides a clean and modern interface for job seekers to explore job opportunites and companies.

## Features
        -Home page
        -Job Listing
        -Company profiles
        -Login Page
        -Register page
        -About page
        =Contact Page

### Technologies Used
      HTML
      css


## Project Structure

```
JobConnect/
│── homepage.html
│── job.html
│── companies.html
│── about.html
│── contact.html
│── login.html
│── register.html
│── design.css
│── images/
```

##  Purpose

This project was developed to practice front-end web development and create a professional recruitment website interface.
